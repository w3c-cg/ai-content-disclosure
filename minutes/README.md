# AI Content Disclosure CG Minutes

This directory contains meeting minutes for the W3C AI Content Disclosure Community Group.

## Conventions
- One folder per meeting: `minutes/YYYYMMDD/`
- Each meeting folder contains a `README.md` with the minutes.
- Meeting join details are shared via the mailing list or calendar invite and are not stored in this repository.
